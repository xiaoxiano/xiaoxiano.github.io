部分电脑配置记录

win10 config

  • git
1git config --global http.proxy http://127.0.0.1:port.http
2git config --global https.proxy https://127.0.0.1:port.http

文件路径：C:\Users\90487\.gitconfig，90487是我的用户名

  • Terminal（临时使用）
1set http_proxy=http://127.0.0.1:port.http          #临时命令行代理
2set https_proxy=http://127.0.0.1:port.http          #临时命令行代理

VM Ubuntu config

  • 代理 Windows中使用ipconfig查看VMware Network Adapter VMnet8的ip地址，ubuntu中进入“设置”-“网络”-“proxy”，在Socks Host里写入之前查到的ip地址

  • git config

1git config --global user.name "John Doe"             #配置github账号用户名
2git config --global user.email johndoe@example.com   #配置邮箱
  • git ssh 代理

创建~/.ssh/config 文件：

1Host github.com
2    User git
3    ProxyCommand nc -v -x ip:port.socks %h %p      #写自己ip

  • 终端代理（临时使用）

    • 方法一：直接在命令前加proxychains

    • 方法二：直接在.bashrc最后加入：

1export http_proxy="http://ip:port.http"          #写自己ip
2export https_proxy="http://ip:port.http"         #写自己ip

1source ~/.bashrc                             #使配置生效

Ubuntu安装go

1sudo apt install snap                   #安装snap，使用snap更易安装、更新

1sudo snap install go --classic          #安装go

1sudo vim /etc/profile                   #编辑profile

1export GOROOT=/snap/go                  #在最后一行添加
2export PATH=$PATH:$GOROOT/bin            

1source /etc/profile                     #保存
2go version                              #验证

1export GO111MODULE=on
2export GOPROXY=https://goproxy.cn       #设置 go get 为国内源（无代理）

Goland

使用

goland无限试用，方法适用IDEA系列

  1. goland中添加第三方插件仓库地址：https://plugins.zhile.io
  2. 安装插件：IDE Eval Reset
  3. 下载
  4. 直接把zip包拖进IDE的窗口来进行插件的安装
  5. Help -> Eval Reset 勾选Auto reset before per restart
