docker

安装docker：

#更新apt包索引
sudo apt-get update

#安装包以允许apt通过 HTTPS 使用存储库
sudo apt-get install \
    apt-transport-https \
    ca-certificates \
    curl \
    gnupg \
    lsb-release
    
#添加Docker官方的GPG密钥
curl -fsSL https://download.docker.com/linux/debian/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg
(此处可能需要安装gpg:sudo apt-get install gnupg2 -y)

#设置稳定存储库
echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/debian \
  (lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
  
#安装最新版本的Docker Engine和containerd
 sudo apt-get install docker-ce docker-ce-cli containerd.io
 
#hello-world 映像验证
 sudo docker run hello-world

卷存储在由 Docker管理的主机文件系统（在 Linux 上的/var/lib/docker/volumes/ ）。卷完全由 Docker 管理，非 Docker 进程不应修改文件系统的这一部分。卷是在 Docker 中持久化数据的最佳方式。

docker compose

安装docker compose：

#下载 Docker Compose 的当前稳定版本
 sudo curl -L "https://github.com/docker/compose/releases/download/1.29.2/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
 
#对二进制文件应用可执行权限
sudo chmod +x /usr/local/bin/docker-compose

#测试安装
docker-compose --version
