购买配件

购买配件只需买比较重要的，有以下两个：

  • mq2 pro 续航版。提升头戴舒适度，闲鱼比较便宜。
  • 磁吸镜片。

网络

解决网络问题一般有以下方式：

  1. 刷路由器，这个方式比较直接，我认为日常不适用的场景较多所以不用。
  2. 电脑热点，下文介绍。

如果你有网线连接电脑，可以试试clash TUN模式开热点。

下面介绍通用的sstap方法。

  1. 打开科学上网，允许局域网连接，国内quest2用户一般有此基础，不再赘述。

  2. 下载sstap，我使用的1.1.0.1版本。

  3. 配置并打开sstap

  4. 打开热点，并打开 网络和Internet设置 -> 更改适配器选项 -> 右键 SSTAP 1 -> 属性 -> 共享
