VPS 选购

发布时间: March 13, 2023 更新时间: December 20, 2023
  • Home
  • computer
  • VPS 选购

购买vps一般需要注意性能、带宽和线路。

  • 性能是根据自身需要的，一般只需要看内存，跑代理512M足够了（考虑到有邻居还是建议1G），跑业务最好在2G以上。

  • 带宽只要搜一下商家的测评，注意共享带宽的实际速度、晚高峰速度就行了。

  • 线路却是一个相对复杂的问题。不仅海内外线路繁多，不同运营商、不同省市的情况也不同，所以最好自己测试（测试方法见下文）。如果不想测，可以去主机测评网等网站看测评，或者电信用CN2、联通用AS9929、移动用CMI。

  • 使用机场最好用IPLC/IEPL线路，几乎不会断连，有名的telegram测评频道前女友们用过的机场可以参考。用机场隐私泄露风险较高。

BGP 边界网关协议，主要是用来连接网络上的独立系统的路由选择协议。主要功能是在传输过程中会选择最优的路由线路。比一般线路要好，但不使用cn2这种最优质线路。

国外运营商

日本 常见线路：NTT、KDDI、IIJ、SoftBank（软银）、Telstra、PCCW

新加坡 常见线路：NTT、Tata通信、Telstra、PCCW、sg.gs

韩国 常见线路：KT、SK、LG

香港 常用线路：PCCW、Telstra、HKBN、HKT、HGC、NTT（香港）、Azure、Google Cloud Platform、Gcorelabs、Tata通信

俄罗斯 常见线路：RosTelecom（俄罗斯电信）、TTK（俄罗斯铁通）、MTS、Megfon、RETN

英国 常见线路：Telia、Cogentco、Level3、Zayo、Tata通信

实际体验（以往返总延迟排序）

  1. 最优：

香港IPLC/IEPL、日本IPLC/IEPL、香港CN2 GIA：价格高。

日本SoftBank（推荐）：约100+ms。

  1. 次优：

新加坡CN2：约200ms，价格稍高。

日本IIJ：约200+ms。

香港CMI：约200+ms。移动网可能好用（我不是），有高峰期。

新加坡NTT：约300+ms。慎买。

美国CN2\美国AS9929（推荐）：约500ms。性价比高，网络审查宽松，且访问的大多服务器设在美国。

  1. 差线路：

racknerd等很廉价，绕路线路，但适合测试用。

优质VPS推荐 BandwagonHost（搬瓦工）：日本软银$169/年（高配）。香港 CN2 GIA最低$899.99/年。

v.ps(需实名):日本软银特价款约39、49欧元/年（经常缺货）。

DMIT：美国CN2特价款$36.90/年（经常缺货）。

ZgoCloud（2021年新商家）：美国AS9929特价款$38.90/年。

vmiss（2022年新商家）：有日本IIJ， 美国CN2、9929的平价大带宽产品。需要注意，他家IP虽然可以解锁大多服务，但使用的是广播ip。

其它卖优质VPS的商家还有Vultr、Linode、digitalocean等。

提示：

  • 机场很可能跑路，用机场一定要月付。

  • 北方地区日本线路表现更好。

  • 有些商家线路会绕路欧美，最好自测。

  • 香港vps：很可能有高峰期，便宜商家的机房容易受到攻击。认准口碑商家，否则不要买。

  • 韩国vps：用过1次，一个月后要求韩国身份实名，不知道是不是都这样。

推荐：美国、新加坡vps表现好价格合适。日本vps极少数小问题，不影响使用。

yum install -y wget unzip && wget https://cdn.ipip.net/17mon/besttrace4linux.zip && unzip best

./besttrace [你的本地ip] -g cn
